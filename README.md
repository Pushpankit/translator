# AI-Powered Language Translator

A desktop-based Language Translator built with Python and Tkinter that supports multiple languages, automatic language detection, translation history, and database integration.

## Features

- Automatic Language Detection
- Multi-language Translation
- User-friendly Tkinter GUI
- SQLite Database Integration
- Translation History Tracking
- CSV Export Functionality
- Copy-to-Clipboard Support
- Translation Statistics

## Tech Stack

- Python
- Tkinter
- SQLite
- Deep Translator
- Pandas
- LangDetect

## Installation

### Clone the Repository

```bash
git clone https://github.com/Pushpankit/translator
cd translator_project
```
### Install Dependencies
pip install deep-translator langdetect pandas

### Run the Application
python trans_gui.py
